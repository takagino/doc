3. リセットCSS

リセットCSSとは、ブラウザがデフォルトで持っているCSSを打ち消すためのCSS。ブラウザ間の表示の差異がなくなり、作業効率も上がるため、実際の現場でも良く使われている。

★『reset css 2020』で検索してみる。

【授業で使用するリセットCSS】
ress.css

★ress.cssの中身を見てみる。

【CDNとは】
CDNとは「Content Delivery Network」の略。同一のコンテンツを多くの配布先、例えば多くのユーザーの端末に効率的に配布するために使われる仕組みのこと。

※制作会社ごとに使用しているリセットCSSが違うため、どのCSSにも対応できる基礎知識が必要。

CSS Flexbox - エンジニアのためのWebチートシート

上記参考サイトの「子要素に指定するプロパティ」については少し難しいので、今の段階で完璧に理解する必要はない。ただし、以下の2点はできるようにしておくこと。

4-1. 横並びにした画像が伸びてしまう問題

【対処法】
「align-items」の値を初期値（stretch）以外に設定する

4-2. 横並びにした要素を固定幅にしたい

【対処法】
.contents {
  display: flex;
}
.main {
  flex: 1;
}
.sub {
  width: 300px;
}
