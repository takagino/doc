「HTML5基礎」01_HTMLとは？

1. HTMLとは？

1-1. HyperText Markup Language

【HyperText（ハイパーテキスト）】
複数の文書を相互に関連付け、結び付ける仕組みのこと。「テキストを超える」という意味から「hyper-（～を超えた）」 「text（文書）」と名付けられた。テキスト間を結びつける参照のことを「ハイパーリンク（略してリンク）」と呼ぶ
すなわち、リンクを介して行き来できるすごいテキスト（書類）のこと
【Markup（マークアップ）】
文章の中に目印（マーク）を付けていくこと
元々の語源は、英語圏の出版業界で著者、編集者、印刷者の間で指示を伝える方法を意味する（校閲）
【Language（ランゲージ）】
言語

まとめると、
HTMLとは「リンクを介して行き来できるすごいテキスト（書類）に目印を付けるための言語」

1-2. 何のための目印？

「何のため？」を知るには、それが出来た歴史（背景）を知るべし！

【WWW（World Wide Web）システムの発明】
1989年、欧州原子核研究機構 (CERN) のティム・バーナーズ＝リーは、「Information Management: A Proposal」（情報管理: 提案）を執筆。
研究状況の確認、論文の資料検索など、数千人にのぼる研究者に必要な情報を効率よく行き渡らせる環境を構築するためのシステムを考案した。
【WWWシステムの実装と公開】
1990年、WWWをより具体化した提案書「WorldWideWeb: Proposal for a HyperText Project」を発表。
HTTP、URL、HTMLなど基幹となる仕組みや、ブラウザ、サーバなどのツールも構築し、人類初のウェブページが公開された。
【W3C（World Wide Web Consortium）の設立】
1994年、WWWで使用される各種技術の標準化を推進する為に設立された非営利団体。ティム・バーナーズ＝リーが創設し、率いている。
W3Cが勧告している規格

1-3. 「WWW = インターネット」ではない

インターネットとは、インターネット・プロトコル(IP)技術を利用して相互接続されたコンピュータネットワーク全体を指し、WWWはその一部の機能でしかない。

【インターネットに含まれる主な技術】
  • telnet
  • 電子メール (SMTP)
  • ファイル転送 (FTP)
  • チャット (IRC)
  • ネットニュース (NNTP)
  • WWW (HTTP)

2. 実際に触ってみよう

2-1. 事前準備

デスクトップが綺麗な人は、仕事が早い！

  1. デスクトップに「HTML5基礎」用のフォルダを作成する
  2. コマごとにデータをわけておく（番号、単元名などをフォルダ名にする「HTML5基礎 / 01_HTMLの基礎」）

2-2. HTMLファイルの準備

  1. 「Visual Studio Code」を立ち上げる
  2. メニュー「File」-「New File」で新規ファイルを作成する
  3. メニュー「File」-「Save」で忘れないうちに保存しておく（ファイル名は「index.html」）
  4. 保存したファイルをブラウザで開いてみる（ファイルをダブルクリック、もしくはブラウザの上にドラッグ）
  • ※ネット上に公開するファイルは、すべてアルファベットで保存すること（文字化けや読み込みエラーの原因になる）
  • ※徐々にショートカットを覚えてください。新規ファイルの作成は「control+ n」、保存は「control + s」（Macの場合は、「control」ではなく「command」）

2-3. 拡張子を表示する

拡張子とは、ファイルの種類を識別するためにファイル名の末尾につけられる文字列。表示されていないと不便なので、常に表示しておくこと。

2-5. 目印をつけてみる

  1. それぞれのテキストを目印で囲む 
    <h1>高木寛貴</h1>
<ul>
<li>1981年1月15日</li>
<li>岐阜県</li>
<li>B型</li>
</ul>
  2. 保存する「control + s」
  3. ブラウザを更新する「control + r」

★どう変化した？

マークアップとは、
「効率よく情報を収集・閲覧するため、文字や文章に目印をつけて、コンピュータが構造や意味を理解できるようにすること」

2-6. 目印の正式名称

  • テキストにつけた目印のことを「タグ」と呼ぶ
  • 特に、先頭のタグを「開始タグ」、末尾のタグを「終了タグ」と呼ぶ
  • タグで囲ったテキスト全体を「要素」と呼ぶ
  • h1のタグを「h1タグ」、h1で囲った要素を「h1要素」と呼ぶ
  • 要素は英語で「element」、タグは「tag」

2-7. タグの作法

  • タグの途中でスペース・改行を入れない
  • 大文字小文字は区別されないが、基本的にはすべて小文字

2-8. どのようなタグがあるのか？

論文のために作られたのだから、論文で考えるべし

【h1〜h6（heading）】
全体の表題や、文章の見出しに使われる
見出しのレベルは順番に使用すること
【p（paragraph）】
文章のかたまり、段落を指定する
【ul（unordered list）、ol（ordered list）、li（list item）】
ナビゲーションメニューや箇条書きなどに使われる。ulは順不同リスト、olは番号順リスト

タグ一覧

よく使用されるタグの意味とスペルを覚えることはもちろん、
「適切なタグ（目印）を付けるために、ページ全体の構造とそれぞれの文章の内容・役割を理解すること」がより重要！

<h1>高木寛貴</h1>
<p>優しくて力持ち</p>
<h2>基本情報</h2>
<ul>
  <li>1981年1月15日</li>
  <li>岐阜県</li>
  <li>B型</li>
</ul>

※要素内の内容が複数行にわたる場合は、「インデント」をつけて見やすくしておくと良い。（ショートカットは「tab」）

3. HTMLの基本構造

<html>
  <head>
  </head>
  <body>
    <h1>高木寛貴</h1>
    <ul>
      <li>1981年1月15日</li>
      <li>岐阜県</li>
      <li>B型</li>
    </ul>
  </body>
</html>

【html要素】
HTML文書の始まりと終わりを指定する
決まり文句のようなもので、全てのHTML文書はhtml要素で囲われている
【head要素】
文書に関する機械可読な情報 （メタデータ）を記述する
題名や制作者名などを記述し、Webブラウザに直接は表示されない
【body要素】
HTML文書の内容を記述する
ここに記述した部分がWebブラウザに表示される

3-1. 最低限のメタデータ

<html>
  <head>
    <meta charset="UTF-8">
    <title>文書のタイトル</title>
  </head>
  <body>
    <h1>高木寛貴</h1>
    <ul>
      <li>1981年1月15日</li>
      <li>岐阜県</li>
      <li>B型</li>
    </ul>
  </body>
</html>

【文字コード】
<meta charset="UTF-8">
文字化けを防ぐために、head要素の上の方（文章が記述される前）に記述する
【title要素】
ウェブページのタイトルを記述する。ブラウザのタブや、検索結果に表示される

4. DOCTYPE宣言

HTMLは常にバージョンアップを繰り返しており、現在の最新バージョンは「HTML5」。バージョンによって多少の違いがあるため、そのファイルがどのバージョンで書かれているかを最初に宣言する必要がある。

<!doctype html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>文書のタイトル</title>
  </head>
  <body>
    <h1>高木寛貴</h1>
    <ul>
      <li>1981年1月15日</li>
      <li>岐阜県</li>
      <li>B型</li>
    </ul>
  </body>
</html>

6. その他の要素と属性

<h1>高木寛貴</h1>
<p>優しくて<br>力持ち</p>
<h2>基本情報</h2>
<ul>
  <li>1981年1月15日</li>
  <li><a href="https://www.pref.gifu.lg.jp/">岐阜県</a></li>
  <li><img src="https://www.motivation-up.com/whats/images/type_b.gif"></li>
</ul>

【br（break）】
文章を改行する
改行については、ここからここまでという範囲は無いので、終了タグが存在しない。このようなタグを「空タグ」という
【a（anchor）】
ハイパーリンクを設定する
「href（hypertext reference）属性」でリンク先を指定する必要がある
【img（image）】
画像を表示する
「src（source）属性」で画像ファイルの場所を指定する必要がある
範囲は無いので「空タグ」

6-1. 属性とは？

  • 要素に設定を付加するもの
  • 開始タグに半角スペースを空け、「属性名」と「属性値」を「=（イコール）」でつないだセットで追加していく
  • 「属性値」は「"（ダブルクォーテーション）」もしくは「'（シングルクォーテーション）」で囲む
  • 複数指定する場合は、さらに半角スペースを空け追加していく
  • 要素ごとに付加できる属性は決められているため、まずは主要なものを覚える
  • 属性は英語で「attribute」
