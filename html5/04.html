3. デザインデータのマークアップ

3-1. マークアップを始める前に

まずはデザインデータを眺めて、どのようにマークアップするか考える

  • 全体の構成は？
  • 見出しの付け方は？
  • 画像になる部分はどこか？（今回はこちらで準備したが、本来は自分で書き出す必要がある）
  • リンクになる部分はどこか？

3-2. HTMLで文書構造を作る

まずは文書構造がきれいに書けるように、HTMLを先に完成させます。（慣れてきたらHTMLとCSSを交互に書いていっても構いません）

見本

【注意点】
  • 文字の大きさなどの見た目に惑わされないこと。（全体の構成と文章の意味だけを考える）
  • class名はまだ付けなくてもよい。
  • AdobeXDの操作に慣れないうちは、いつのまにかデザインを変更してしまうことが多々あるので、データを複製してバックアップをとっておくと良い。
【AdobeXDで文字をコピーする】
「選択ツール」で文字をクリックしても選択できるが、今回のデータは細かくグループ化されており、順番にダブルクリックしていく必要がある。
その場合は「文字ツール」に切り替えて文字をクリックすると、直接選択できて便利。
「文字ツール」で選択した文字をダブルクリックすることで内容が選択されるので、コピーをする。

※文字全体の選択が解除されてしまった場合は、「ctrl + a（Macは、command + a）」で全選択ができる。

4. スタイルの継承

上にある要素から順番にCSSをつけていってもページは完成するが、マークアップで重要なことは、いかに早く、効率よく、わかりやすく、修正しやすくCSSを書くかどうか。

デザインデータをもう一度良く見ると、見出しも本文もほぼ同じ文字色で、本文のフォントサイズもほぼ同じ大きさなのがわかる。同じ値のプロパティを要素ごとに書いていては無駄なので、スタイルが「継承」されるプロパティは、親要素に一括で書いた方が効率が良い。

4-1. スタイルの継承とは？

親要素のプロパティの値が、子要素に引き継がれること。プロパティによって継承されるものとされないものに分かれる。

【継承される主なプロパティ】
  • color
  • font-size
  • font-style
  • font-weight
  • letter-spacing
  • line-height
  • list-style
  • text-align　など

CSSを書くときにまずやることは、継承の概念を利用し、bodyに共通のプロパティを指定して、一番多いであろうテキストの値に合わせておくこと！

【大企業のサイトの「body」にどんなプロパティが指定してあるか？】
  • Apple
  • docomo
  • au
  • Softbank

5. 要素をまとめるタグ「div」

上にある要素から順番にCSSをつけていってもページは完成するが、マークアップで重要なことは、いかに早く、効率よく、わかりやすく、修正しやすくCSSを書くかどうか。

完成版をもう一度良く見ると、すべての要素の横幅がウィンドウサイズに合わせて伸縮し、ある一定の大きさ以上で固定される作りになっている。同じ値のプロパティを要素ごとに書いていては無駄なので、要素をひとつのグループにまとめ、一括で書いた方が効率が良い。

【div（divition）】
  • 区分・分割を意味する。
  • div要素自体に特別な意味はなく、要素をグループ化する目的で使う。
  • ページ内で多数使用する可能性が高いため「class名」をつけて区別しておく必要がある。

<!doctype html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Company</title>
    <link rel="stylesheet" href="css/style.css">
  </head>
  <body>
    <!--
    全体を <div> で囲む。
    class名「wrapper」は「包むもの」という意味で、全体を囲むときによく使われる。
    -->
    <div class="wrapper">
      <h1><img src="images/img_logo.png"></h1>

      <img src="images/pic_mainvisual.png">
        ・
        ・
        中略
        ・
        ・
      <p>2019 All Rights Reserved.</p>
    </div>
  </body>
</html>
.wrapper{
  width: 90%;
  max-width: 750px;
  margin: 0 auto;
}

【max-width】
  • 要素の最大幅を設定する。
  • 単位や初期値
【min-width】
  • 要素の最小幅を設定する。
  • 単位や初期値

※ボックスモデルを可視化して、仕組みを今一度復習しておくこと。

6. 「AdobeXD」の使い方

6-1. 余白の長さの調べ方

  • 「選択ツール」で調べたいオブジェクトをクリックし「alt」キーを押すと、アートボードの上下左右までの長さを表示
  • 「選択ツール」で調べたいオブジェクトをクリックし、別のオブジェクトをマウスオーバーした状態で「alt」キーを押すと、オブジェクト間の長さを表示

6-2. テキスト関連の数値の調べ方

【line-heightの計算】
フォントサイズが「24」、行の高さが「36」の場合、「36÷24=1.5」

※GoogleChromeのURL欄に「36/24」と入力して検索するのが早い。
