3. スマホ対応の基礎

3-1. スマホ対応の必要性

  • 全世代で「スマホからのネット利用」がトップに
  • 「モバイルファースト・インデックス」の開始

3-2. スマホ対応の方法「別個のURL」

別々のURLでサイトを構築したり、何らかの方法でPCとスマホを自動的に判別し表示するファイルを振り分ける方法。

  • Facebook（PC版・スマホ版）
  • yahoo
  • amazon
【メリット】
スマホで閲覧することに特化したWebサイトを構築できる
古いブラウザを考慮する必要がないため、HTML5やCSS3などの最新技術をふんだんに使うことができる
【デメリット】
更新や追加・修正の際に2倍の手間が掛かる
「単にコピーしただけのWebサイト」という誤認識を受け、Googleなどの検索エンジンで検索結果の上位に表示されにくくなる恐れがある

3-3. スマホ対応の方法「レスポンシブウェブデザイン」

ユーザーのデバイスの種類（パソコン、タブレット、モバイル、非視覚的ブラウザ）に関係なく、同じ「URL」で同じ「HTMLコード」を配信しつつ、画面サイズに応じて表示を変更する方法。

  • RWDJP
  • Media Queries
【メリット】
更新などの際に複数のHTMLファイルを書き換える必要がない
WebサイトのURLが統一されているため、シェアやリンクが容易
【デメリット】
スマホでの表示に時間がかかりやすくなってしまう
最初の設計や構築が複雑

Googleは「レスポンシブウェブデザイン」をおすすめしている。

4. レスポンシブウェブデザインの基礎

4-1. 5つのポイント

  • Fluid Layout（フルードレイアウト）
  • Fluid Image（フルードイメージ）
  • Viewport（ビューポート）
  • Media query（メディアクエリ）
  • Breakpoint（ブレイクポイント）

4-2. Fluid Layout（フルードレイアウト）

Webページのレイアウトは大きく分けると2種類ある。

【Fixed Layout（フィックスドレイアウト）】
width, margin, padding を絶対単位の px で指定する
【Fluid Layout（フルードレイアウト）】
width, margin, padding を相対単位の % や em などで指定する
【固定（px）から可変（%）へ変換】
変換したい値（要素や余白の幅）÷ 変換したい値の親要素の幅 x 100

4-2. Fluid Image（フルードイメージ）

img {
  max-midth: 100%;
}

4-3. Media query（メディアクエリ）

画面サイズや解像度などの閲覧環境に応じて適用するスタイルを切り替えられるCSS3の機能。

【CSSに書き込む方法】
/* 共通のスタイルを書く */

body{ background-color: #fff; }

/* ウィンドウサイズが600px〜1000px */

@media (min-width:600px) and ( max-width:1000px) {
  body{ background-color: #f00; }
}

/* ウィンドウサイズが1001px以上 */

@media (min-width:1001px) {
  body{ background-color: #00f; }
}

※単語ごとのスペースを忘れないこと

【link要素として記述する方法】
<link rel="stylesheet" href="style.css">
<link rel="stylesheet" href="style_tablet.css" media="(min-width:600px) and (max-width:1000px)">
<link rel="stylesheet" href="style_pc.css" media="(min-width:1001px)">
【メディアクエリを並べる順番】
/* モバイルファースト
-----------------------------------*/
body{ background-color: #fff; }

/* ウィンドウサイズが600px以上の時 */
@media (min-width:600px) {
  body{ background-color: #f00; }
}

/* ウィンドウサイズが960px以上の時 */
@media (min-width:960px) {
  body{ background-color: #00f; }
}


/* デスクトップファースト
-----------------------------------*/
body{ background-color: #fff; }

/* ウィンドウサイズが960px以下の時 */
@media (max-width:960px) {
  body{ background-color: #f00; }
}

/* ウィンドウサイズが600px以下の時 */
@media (max-width:600px) {
  body{ background-color: #00f; }
}
